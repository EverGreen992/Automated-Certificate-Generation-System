# Automated Certificate Generation System using Google Sheets, Autocrat & Docs
## Project Overview:
This Project is an automated certificate generation system built using Google Workspace tools. It generates personalized training certificates based on participant data stored in Google Sheets. The System uses logic to automatically generate certificates only for participants who have completed the course and have not yet received their certificate
## Problem Statement:
Manual certificate creation is time-comsuming and prone to error.
This project solves the problem by automating:
- Certificate generation
- Data merging
- Conditional filtering
- Output file organization
## Tools Used:
- Google Sheets (Data management)
- Google Docs (Certificate template design)
- Smart Chimps (Dynamic references)
- Tables (Structured formatting)
- Autocrat Add-on (Mail merge automation)
- PDF export for professional certificate output
